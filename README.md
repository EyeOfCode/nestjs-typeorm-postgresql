## Description

Exp nestjs typeorm postgresql

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## lib

```bash
# postgres typeorm
$ npm install --save @nestjs/typeorm typeorm pg

# dotenv
$ npm install --save dotenv

# nest js config
$ npm i --save @nestjs/config

# yup
$ npm i --save @nestjs/config
$ npm install -S yup

# graphql
$ npm i @nestjs/graphql graphql-tools graphql apollo-server-express
```
